# Full Stack Demo
This is a beginner-friendly demo that shows how the frontend, backend, and database of a web application work together. Youâ€™ll build a tiny app where users can send and view messages.

## Setup Instructions
1. Clone the Repo
```
git clone <your-repo-url>
cd fullstack-demo
```

2. Start the Backend (Flask)
```
cd backend
pip install flask flask-cors
python app.py
```

Backend runs at http://127.0.0.1:5000

3. Start the Frontend (React)
```
cd frontend
npm install
npm run dev
```

Frontend runs at http://localhost:5173
